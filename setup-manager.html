do setup completo
Baixar projeto FINAL no Mac Mini 2
cd ~ && unzip jarvis-claude-FINAL.zip -d jarvis-claude
Verificar Node.js v22+
node --version
Se menor que v22: nvm install 22 && nvm use 22
Instalar dependências
cd ~/jarvis-claude && npm install
Instalar Playwright + Chromium
chmod +x setup-playwright.sh && ./setup-playwright.sh
Configurar .env com as keys (esta página!)
Gerar .env → copiar → nano ~/jarvis-claude/.env
Criar bot no @BotFather do Telegram
/newbot → escolher nome @jarvisclaude
Copiar o token e adicionar no campo Telegram abaixo
Testar Jarvis
node src/index.js
Deve aparecer "🤖 Jarvis online" e "🌐 Webhook Server"
Rodar em background com pm2
npm install -g pm2 && pm2 start src/index.js --name jarvis && pm2 save && pm2 startup
Testar no Telegram: /start
Abrir @jarvisclaude → /start
Deve responder com a mensagem de boas-vindas do Jarvis
